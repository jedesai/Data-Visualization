'# Data-Visualization'

Seaborn graphs are broadly divided into six categories:

1. Relational Plot

- Scatter
- line

2. Distribution Plot

- histogram
- kde
- rug

3. Categorical Plot

- bar
- count
- box
- violin
- swarm
- strip

4. Regression Plot

5. Matrix Plot

- heatmap
- clustermap

6. Residual Plot
7. Multi Plot
